# 🌡️ Temperature Converter (Java)

## 📌 Overview

This project is a simple **Temperature Converter** developed in Java as part of the **Saiket Systems Internship – Task 2**.
It allows users to convert temperatures between **Celsius and Fahrenheit** using a console-based interface, with an option to exit the program.

---

## 🚀 Features

* Convert **Celsius to Fahrenheit**
* Convert **Fahrenheit to Celsius**
* User-friendly **command-line interface**
* Uses **core Java concepts** (methods, user input, switch-case)
* Accurate mathematical calculations

---

## 🛠️ Technologies Used

* Java
* Scanner Class (for user input)

## ▶️ How to Run

### Step 1: Clone the Repository

```
git clone https://github.com/hewageumesha/Java-Temperature-Converter.git
```

### Step 2: Navigate to Project Folder

```
cd java-temperature-converter
```

### Step 3: Compile the Program

```
javac TemperatureConverter.java
```

### Step 4: Run the Program

```
java TemperatureConverter
```

---

## 🧠 How It Works

1. User selects an option:

   * 1 → Celsius to Fahrenheit
   * 2 → Fahrenheit to Celsius
2. User enters temperature value (if applicable)
3. Program calculates and displays the result or exits

---

## 🔢 Conversion Formulas

* Celsius to Fahrenheit:

  ```
  (C × 9/5) + 32
  ```
* Fahrenheit to Celsius:

  ```
  (F − 32) × 5/9
  ```
